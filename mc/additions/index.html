This is my resource pack project based around adding new cosmetic things into the game and making the existing things better.
Currently it changes almost all of the UI, replaces the default pixel font with Arial using Minecraft's vanilla font capabilities. It also makes some things that were obtrusive, for example shields, use less screen space and yet look just as good.
